吴恩达 线性回归 梯度下降 代价函数
机器学习
发布日期:   2021-10-25
文章字数:   1.9k
阅读时长:   7 分
阅读次数:  

接下来所要学习的第一个算法是线性回归算法，通过这个算法，可以了解监督学习的完整流程。

首先声明回归问题中常用的标记解释如下：

$m$ 表示训练集中实例的数量

$x$ 表示特征/输入变量

$y$ 表示目标变量/输出变量

$(x,y)$ 代表训练集中的实例

$\left(x^{(i)}, y^{(i)}\right)$ 代表第$i$个实例

$h$ 代表学习算法的解决方案或者称为假设$(hypothesis)$

$h$是一个从$x$到$y$的函数映射。当$h_{\theta}(x)=\theta_{0}+\theta_{1} x$时，因为只含有一个特征/输入变量，因此该问题被称为单变量线性回归问题。

代价函数

假设函数中我们设定的参数决定了我们得到的直线相对于我们的训练集的准确程度，模型所预测的值与训练集中的实际值之间的差距，被称为建模误差(modeling error).

我们的目标是选择出能够最小化建模误差平方和的模型参数，即使得代价函数$J\left(\theta_{0}, \theta_{1}\right)=\frac{1}{2 m} \sum_{i=1}^{m}\left(h_{\theta}\left(x^{(i)}\right)-y^{(i)}\right)^{2}$最小。该代价函数也被称作平方误差函数，有时也被称为平方误差代价函数。我们需要的是一种有效的算法，能够自动找出使代价函数取最小值的参数。

梯度下降

梯度下降是用来求函数最小值的算法，开始时随机选择一个参数的组合$(θ{0},θ{1},……θ_{n})$，计算代价函数，然后寻找能够让代价函数值下降最多的参数组合，持续迭代知道寻找出局部最小值。因为并没有尝试完所有参数组合，所以不能确定当前局部最小值是否是全局最小值，当我们选择不同的初始参数组合时，会得到不同的局部最小值。

批量梯度下降(batch gradient descent)算法的公式为:$\theta_{j}:=\theta_{j}-\alpha \frac{\partial}{\partial \theta_{j}} J\left(\theta_{0}, \theta_{1}\right) \quad(\text { for } j=0 \text { and } j=1)$

其中，$α$是学习率，决定了下一步迈出的步子有多大，在批量梯度下降中，每一次都同时让所有的参数减去学习率乘以代价函数的导数。

如果$α$太小，会导致下降过程只能像小宝宝一样一点点地挪动，这样就只能走很多步才能到达最低点。如果$α$太大，那么梯度下降法可能会越过最低点，甚至可能无法收敛，离最低点越来越远，乃至发散。

如果恰好将$\theta_{1}$设置在了局部最低点，此时已经是一个局部最优解，那么$\theta_{1}$不再改变，梯度下降法其实什么也没错，不会改变参数的值。

随着梯度下降法的运行，移动的幅度会自动变得越来越小，直到最终移动幅度非常小，此时会发现，已经收敛到局部极小值。所以，梯度下降法将会自动采取最小的幅度，这就是梯度下降法的做法，所以实际上没有必要再另外减小$\alpha$。

以上便是梯度下降算法的基本原理，可以用来最小化任何代价函数，而不仅仅局限于线性回归中的代价函数。

批量梯度下降算法这个名字就说明了，我们在运算中，每次都需要考虑所有的训练样本，但事实上，也有其他类型的梯度下降算法并不是批量型的，不考虑整个数据集，而是每次都只关注训练集中的一些小的子集。这个方法是正规方程法(normal equations)，再数据量较小的情况下，正规方程法比梯度下降法要更适用一些。

多变量线性回归

多维特征

在对房价预测的模型中，加入更多的特征，将会更成一个含有多个变量的模型。

支持多变量的假设函数$h$表示为$h_{\theta}(x)=\theta_{0}+\theta_{1} x_{1}+\theta_{2} x_{2}+\ldots+\theta_{n} x_{n}$。

这个函数中有$n+1$个参数和$n$个变量，为了简化公式，使$x_{0}=0$，则公式转化为$h_{\theta}(x)=\theta_{0} x_{0}+\theta_{1} x_{1}+\theta_{2} x_{2}+\ldots+\theta_{n} x_{n}$.此时的参数维数与训练实例维数都是$n+1$维，因此公式可以转化为$h_{θ}(x)=θ^TX$.

多变量线性回归中，同样需要构造代价函数，$J\left(\theta_{0}, \theta_{1} \ldots \theta_{n}\right)=\frac{1}{2 m} \sum_{i=1}^{m}\left(h_{\theta}\left(x^{(i)}\right)-y^{(i)}\right)^{2}$

建模的目标仍然是找出使得代价函数最小的一系列参数。

特征缩放

在面对多维特征的时候，要帮助这些特征，是它们具有相近的尺度，从而使得梯度下降算法更快的收敛。具体做法是让特征缩放到-1到1之间。

学习率

梯度下降算法收敛所需要的迭代次数根据数据模型的不同而不同，我们可以绘制迭代次数和代价函数的图表，来观测算法在何时趋于收敛。

梯度下降算法的每次迭代受到学习率的影响，如果学习率𝑎过小，则达到收敛所需的迭代次数会非常高；如果学习率𝑎过大，每次迭代可能不会减小代价函数，可能会越过局部最小值导致无法收敛。

特征和多项式回归

在房价预测问题中：

$h_{θ}(x)=θ{0}+θ{1}frontage+θ_{2}depth$

另$x=frontagedepth$，则$h_{θ}(x)=θ{0}+θ{1}x$

有时候线性回归并不适用于所有数据，此时需要曲线来适应我们的数据，比如一个二次方程$h_{θ}(x)=θ{0}+θ{1}x+θ_{2}x^2$，或者三次方模型$h_{θ}(x)=θ{0}+θ{1}x+θ_{2}x^2+θ_{3}*x^3$.

值得注意的是，在使用多项式回归模型时，在运行梯度下降算法之前，非常有必要进行特征缩放。

正规方程

对于某些线性回归问题，正规方程方法是更好的解决方案。

正规方程是通过求解下面的方程来找出使得代价函数最小的参数的： $\frac{\partial}{\partial \theta_{j}} J\left(\theta_{j}\right)=0$.

假设我们的训练集特征矩阵为 $X$（包含了 $x_{0}$ = 1）并且我们的训练集结果为向量 $y$，则利用正规方程解出向量 $𝜃 = (𝑋^T𝑋)^{-1}𝑋^T𝑦$。

然而，对于那些不可逆的矩阵（往往是由于特征之间相互不独立，或者特征数量大于训练集的数量），正规方程的方法是不能用的。

只要特征变量的数目并不大，标准方程是一个很好的计算参数𝜃的替代方法。
具体地说，只要特征变量数量小于一万，我通常使用标准方程法，而不使用梯度下降法。当讲到分类算法，像逻辑回归算法，并不能使用标准方程法。对于那些更复杂的学习算法，将不得不仍然使用梯度下降法。因此，梯度下降法是一个非常有用的算法，可以用在有大量特征变量的线性回归问题。
