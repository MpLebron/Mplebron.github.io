PyQt简介

Python图形界面开发的几种方案

程序的用户交互界面，英文称之为UI(user interface)。

用Python语言开发跨平台的图形界面程序，主要有三种选择：

库 优点： 缺点
Tkinter 是Python标准库，稳定，发布程序较小 控件少
wxPython 控件丰富 稳定性较差，文档少，用户少
PySide2，PyQt5 控件丰富，跨平台体验好，文档完善，用户多（问题好解决） 库较大，发布程序较大

库的选择：

  • 如果开发小工具，界面简单，选择Tkinter
  • 如果发布功能较多的正是产品，选择基于Qt的PySide2，PyQt5

PySIde2，PyQt5简介

PySide2,PyQt5都是基于Qt库。Qt是一个功能非常强大的图形界面开发库，底层使用C++语言开发，PySide2，PyQt5使我们可以通过Python语言使用Qt。

两个库的使用差别很小，调用的接口几乎一模一样。

安装PySide2

pip install pyside2

如果程序要发布给客户使用，可以选择32位的Python解释器，这样打包发布的exe程序可以兼容32位的Windows。

安装PyQt5

pip install pyqt5-tools

可以同时安装PyQt5和一些重要的工具，比如Qt designer。

案例开发

界面设计

开发一个程序，使用户输入一段文本，包含：姓名，薪资，年龄。可以把薪资在 2万 以上、以下的人员名单分别打印出来。

格式如下：

from PySide2.QtWidgets import QApplication, QMainWindow, QPushButton, QPlainTextEdit

app = QApplication([])

window = QMainWindow()
window.resize(500, 400)
window.move(300, 310)
window.setWindowTitle('薪资统计')

textEdit = QPlainTextEdit(window)
textEdit.setPlaceholderText("请输入薪资表")
textEdit.move(10,25)
textEdit.resize(300,350)

button = QPushButton('统计', window)
button.move(380,80)

window.show()

app.exec_()

其中：

  • QApplication提供了整个图形界面程序的底层管理功能，比如界面初始化，程序入口参数，用户事件处理等等……因此，在任何界面控件对象创建之前，都要首先创建QApplication对象。
  • QMainWindow，QPlainTextEdit，QPushButton使三个控件类，分别对应界面的主窗口，文本框和按钮，都是控件基类对象QWidget的字类。要在界面上创建一个控件，就需要创建这个空间所对应类的一个实例对象。
  • Qt中，控件(Widget)是层层嵌套的，都有一个参数window，比如，QPlainTextEsit(window),QPushButton('统计',window)。但是在实例化QMainWindow主窗口时，却没有指定父类，因为它已经是最上层的控件了。
  • 控件对象的move(x,y)方法决定了控件显示的位置，是窗口左上角坐标相对于屏幕左上角横坐标x个像素，纵坐标y个像素的位置。
  • 控件对象的resize(x,y)决定了窗口的宽度和高度分别为x，y个像素。
  • 必须加上window.show()才能将控件都显示在界面上。
  • 最后，通过app.exec_()进入QApplication的事件处理循环，接受用户的输入事件，并分配给相应的对象处理。

界面动作处理(signal和slot)

实现了界面搭建后，需要针对用应用需求进行开发。实现：当用户点击统计按钮时，从界面控件QPlainTextEdit里面获取用户输入的字符串内容。

在Qt系统中，当界面上的一个控件被操作时（点击，输入文本，鼠标拖拽等），就是发出signal（信号），表明某个事件发生了。我们需要预先在代码中指定处理这个signal的函数，这个函数叫做slot。

from PyQt5.QtWidgets import QApplication,QMainWindow,QPushButton,QPlainTextEdit,QMessageBox

def handleCalc():
    info = textEdit.toPlainText()

    # 薪资20000以上和20000以下的名单
    salary_above_20000 = ''
    salary_below_20000 = ''
    for line in info.splitlines():
        if not line.strip():
            continue
        parts = line.split(' ')
        # 去掉列表中的空字符串内容
        parts = [p for p in parts if p]
        name,salary,age = parts
        if (int(salary)) > 20000:
            salary_above_20000 += name+'\n'
        else:
            salary_below_20000 += name + '\n'
    QMessageBox.about(window,'统计结果',f"""薪资20000以上的有:\n{salary_above_20000}\n薪资20000以下的有:\n{salary_below_20000}""")

app = QApplication([])

window = QMainWindow()
window.resize(500,400)
window.move(300,310)
window.setWindowTitle('薪资统计')

textEdit = QPlainTextEdit(window)
textEdit.setPlaceholderText("请输入薪资表")
textEdit.move(10,25)
textEdit.resize(300,350)

button = QPushButton('统计',window)
button.move(380,80)
button.clicked.connect(handleCalc)

window.show()

app.exec_()

其中：

button.clicked.connect(handleCalc)

该行代码的意义是：当button被点击（clicked）后，连接（connect）handleCalc这样一个slot上。
